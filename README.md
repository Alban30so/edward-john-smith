# edward-john-smith
Navigateur web I5 Architecture logiciel

Edward John Smith était le navigateur en chef du Titanic, il s'agit donc d'un navigateur paraissant fiable mais qui ne l'était pas tant. Je vous laisse tirer les conclusions quant à ce navigateur nouvelle génération, résistant aux isberg (si si tkt)

La fonctionalité de gestionnaire de mot de passe s'utilise de la façon suivante : 
- enregistrez vos identifiant via l'interface dans la toolbar
- une fois sur le site web (github.com par exemple), si vous faites un clic droit sur la zone de saisi du MOT DE PASSE et que vous avez un identifiant associé à ce site, alors vous verrez un menu permettant de remplir automatiquement vos identifiants.

Les données sont sauvegardées dans un fichier passwords.json créé lors de la première utilisation du gestionnaire de mot de passe.
